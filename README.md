# 行测速练小程序

行测速练是一个面向行测备考的微信小程序，核心能力是“专项训练 + 结构化笔记”。

## 小程序能力

### 训练

- 支持资料分析、基础计算等专项训练入口。
- 支持按训练目录进入题组练习。
- 支持训练会话、答题过程和结果页展示。
- 支持本地记录训练状态与结果数据。

### 笔记

- 支持行测知识点结构化学习。
- 学习链路为：笔记首页 → 单科知识地图 → 分类知识点列表 → 知识点详情。
- 已整理资料分析、数量关系、判断推理、言语理解、图形推理 5 个模块。
- 知识点详情支持识别信号、规则总结、技巧提示、例题讲解、结构导图和局部示例图。

### 图形推理知识库

- 支持图形推理 6 大分类、28 个知识点。
- 支持 `mind-map` 结构化导图展示。
- 支持局部示例图懒加载和加载失败兜底。
- 内容以结构化知识卡为主，不展示整页 PDF 原图。

### 认知训练

- 支持舒尔特方格、闪算、闪记等认知训练模块。
- 支持认知训练引导页和训练会话页。
- 支持训练最佳记录的本地存储。

### 页面与体验

- 底部 Tab 保留训练和笔记两个核心入口。
- 统一使用主题样式和 UI 基础样式。
- 图片资源支持本地路径、云资源前缀和占位图兜底。
- 知识详情页支持分块懒加载，降低长内容首屏压力。

## 本地校验

```bash
node scripts/validate-notes.js
```

校验通过时，知识库数据应无 `problems` 和 `warnings`。
